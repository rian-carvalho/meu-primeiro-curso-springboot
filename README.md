# 🚀 Meu Primeiro Projeto com Java Spring Boot

Este repositório representa meu primeiro contato prático com o ecossistema **Java + Spring Boot**.

O projeto foi desenvolvido acompanhando um curso completo no YouTube, onde implementei todas as funcionalidades **codando junto com o professor**, mas com um diferencial importante:
👉 **refiz cada parte do código por conta própria e estudei o funcionamento de cada componente.**

---

## 🧠 Sobre meu aprendizado

Durante o desenvolvimento, eu:

* Reescrevi os códigos para fixar o conteúdo
* Analisei a função de cada camada do projeto
* Comecei a entender como funciona uma API REST na prática
* Tive meu primeiro contato com boas práticas do Spring Boot

⚠️ **Importante:**
Apesar de já compreender diversos conceitos, ainda estou em processo de evolução e aprofundamento. Este projeto faz parte da minha jornada como desenvolvedor.

---

## 📚 Conteúdos estudados

* Estrutura do **Spring Boot**
* Criação de projetos com **Spring Initializr**
* Arquitetura em camadas (**Controller, Service, Repository, Model**)
* Persistência de dados com **Spring Data JPA**
* Integração com **PostgreSQL**
* Criação de **API REST**
* Tratamento de **exceções**
* Implementação de **exceções globais**
* Introdução à **segurança com Spring Security**
* Autenticação utilizando **JWT**
